geom_tile

This is a simple geom_tile example.



Steps:

  • The Html part of the code just creates a div that will be modified by d3 later on.

  • The first part of the javascript code set a svg area. It specify the chart size and its margin. Read more.

<script>
	const data = [
		{ label: "A", category:1, value: 1, value2: 5, value3: 15},
	]

	const aes = {
		x: "category",
		y: "value2",
		width: "value",
		height: "value3"
	}

		let g = new H.gg(data, aes);
		g.geom_tile({size:2, color:'blue'})
		g.xlim([0,11])
		g.ylim([0,25])
		g.theme_dark();
		g.print('#my_view');

</script>
